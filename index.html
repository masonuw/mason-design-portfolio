Project 1: Send News

This is my first project as a student in the UW Software Development Bootcamp, where me and several other students developed a website that both pulled news articles and allowed for users of the website to have a text-to-speech API read the news article back to you. It is intended for people that want to keep up with the news, but are unable to take the time to fully sit down and read an article.
